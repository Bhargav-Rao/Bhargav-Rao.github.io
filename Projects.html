PROJECTS
  • Autoscaling of Cloud Foundry
    (Guide: Dr Dinkar Sitaram, Head, Centre for Cloud Computing, PESIT, Feb'15 - May'15)
    • Objective : To propose a better technique to autoscale Cloud Foundry using predictive approaches.
    • The project involves adding a Perceptron based predictive scaling algorithm to BOSH.
  • Calculation of the diameter of Zone of Inhibition
    (Guide: Prof Mahendra M Nayak, Crucible of Research and Innovation, PESIT, May'14 - Feb'15)
    • Objective : To calculate the diameter of a particular zone of inhibition of antibacterials using OpenCV.
    • The project is a multi-disciplinary project involving detailed programming using OpenCV.
    • The project is aimed at the rural hospitals, which are financially backward (as the present device which does the same work is ten times the cost of the final product of this project).
  • Extracting information and Visualization of LAK Dataset
    (Guide: Prof Nitin V Pujari, Dept of Computer Sciences, PESIT, Jul'14 - Nov'14)
    • Objective : To extract information from the LAK-Dataset and visualize it using Cypher Text.
    • To analyse the contents of the LAK-Dataset, convert it into a Graph DB and provide means of querying.
    • The final project will be of great help to those researchers who will analyze the LAK Dataset.
