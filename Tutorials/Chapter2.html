Chapter 2 Basic Functions

Predicates return true or false, ends with a p 
 print numberp 1 ; displays if 1 is a number
Similarly there are functions like wordp, listp etc ...
List
Lists are ordered collection of any len, arrays are of specific len 
 print [a list]
 print {an array}
array 4 , creates an array of four lists
Make and Thing
make keyword is used to create variables, not strongly typed, and not case sensitive 
 make "num1 1; stores 1 in num1
 make "num2 "Hi; stores hi in num2
thing keyword is used to access variables 
 print thing "num1
else we can also use : 
 print :num2
