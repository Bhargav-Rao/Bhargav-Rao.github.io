Chapter 2 Basic Functions

Make and Thing
make keyword is used to create variables, not strongly typed, and not case sensitive 
 make "num1 1       ; stores 1 in num1
 make "num2 "Hi     ; stores hi in num2
thing keyword is used to access variables. We can also use : instead of thing 
 print thing "num1
 print :num2
String Functions
The function count returns the length of a string. The function first returns the first letter of a string and but first returns all other letters. The function item takes two inputs, a number and a word and returns the character at the position in the word. The function word concatenates two strings. 
 make "name "LOGO        ; Create a string
 print count :name       ; Print the length of the character
 print first :name       ; Print first character
 print butfirst :name    ; Print all the other characters
 print item 2 :name      ; Print the second character
 print word "hello :name ; Concat 'hello' to name (NOT INPLACE)
sentence is used to make a string out of words. It can be used as follows. (Note that without the parenthesis it is the same as word) 
 (sentence :name "is "fun "to "learn)
