Chapter 3 Procedures

Syntax to Create a procedure 
 define "name [[List of parameters] [instruction instruction ...]]
Examples 
 define "newline [[][print "]]
define "Twogaps [[][newline newline]]
define "add [[a b][print sum :a :b]]
They can be called like the builtin procedures. 
 add 2 3
The to keyword is used to create new methods 
to mul :a :b
    print product :a :b
end      ; there is an end but no begin ...
Now to call the procedure 
 mul 2 3
to is the same as define but the latter is inline way of declaring functions.
