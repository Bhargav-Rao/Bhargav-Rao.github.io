Returning values from a function
To return values we use output 
 to fun 
    output "|This is fun|
 end
 
 print fun
Recursive functions
Recursive functions can be written by calling the function within it's body. Example, to calculate the factorial of a number: 
 to fact :n
    ifelse equalp :n 0 [output 1][output product :n fact difference :n 1]
 end
 
 print fact 10
