Chapter 1 Introduction

Displaying outputs
print is a function used to print anything on the console. For Example 
 print "Hello 
 print "|Hello World|
Comments can be made using the ;. All those statements written after the semicolon is ignored by the interpreter. For Example 
 ; This is a comment 
 ; This want be executed
Predicates
Predicates return true or false, ends with a p 
 print numberp 1 ; displays if 1 is a number
Similarly there are functions like wordp, listp etc ...
Lists and Arrays
Lists are ordered collection of any length, arrays are of specific length 
 print [a list]
 print {an array}
The statement array 4 , creates an array of four lists. Likewise, arraytolist and listtoarray are conversion functions
