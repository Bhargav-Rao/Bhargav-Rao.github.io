Chapter 4 Procedures Part 2

Returning values from a function
To return values we use output 
 to fun 
    output "|This is fun|
 end
 
 print fun
Recursive functions
Recursive functions can be written by calling the function within it's body. Example, to calculate the factorial of a number: 
 to fact :n
    ifelse equalp :n 0 [output 1][output product :n fact difference :n 1]
 end
 
 print fact 10
Local and Global Scope
The local keyword makes the variables local to the function 
 to makealocal
     local "num1
     make "num1 3     ; here num1 is local variable
 end                  ; No longer accesible outside makealocal procedure
Similarly the global keyword is used to make the variables global. However it is not required as all variables defined within a function are in the global scope.
Note On defining a function, the global variables inside are not defined. They are defined only after you have called the function.
